## Installation and Setup Instructions
Clone down this repository. You will need `node (v.14.15.5)` and `yarn (v.1.22.5)` installed globally on your machine.

Installation:
`yarn install`

To Start Server:
`yarn dev`

To Visit App:
`localhost:3000`
